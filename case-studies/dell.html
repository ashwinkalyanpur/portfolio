Dell – Network Infrastructure Management

Role UX Designer
Duration 7 Months
Team Product Owner, Infrastructure Engineers, UX
Tools Figma, Photoshop, Azure DevOps, Jira

Designed network infrastructure dashboards for monitoring VM status, connectivity validation and router/switch configuration processes.

Project Overview

At Dell Technologies, the goal was to design an enterprise platform that allowed engineering teams to monitor infrastructure devices such as routers, switches, and virtual machines across distributed systems.

Problem

Infrastructure engineers relied on multiple fragmented tools to track device connectivity and system health. This created delays in identifying network issues and increased troubleshooting time.

My Role

  • Designed monitoring dashboards for network infrastructure
  • Conducted workflow analysis with engineering teams
  • Created wireframes and interactive prototypes
  • Presented UX solutions during weekly design reviews

Design Process

  • Mapped infrastructure monitoring workflows
  • Designed dashboard layouts for device health monitoring
  • Validated designs with engineering stakeholders

Solution

Developed unified monitoring dashboards providing engineers with real-time visibility into system performance, device connectivity, and network health.

Impact

  • Improved infrastructure visibility
  • Reduced troubleshooting time
  • Enabled faster identification of network failures
